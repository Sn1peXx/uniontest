Устаревшие версии ОС

Windows XP / OS X 10.6

Окончание поддержки Windows XP и Mac OS 10.6 (или ниже) Компания Microsoft прекратила поддержку операционной системы Windows XP в апреле 2014 года. Это означает, что Microsoft больше не будет поставлять обновления для системы безопасности и ошибок, поэтому любой компьютер, использующий эту операционную систему, может быть подвержен риску заражения вредоносными программами и вирусами.

UNION был вынужден вывести из эксплуатации свои компьютеры с Windows XP для поддержания безопасности учебного заведения, поэтому сотрудники, находящиеся в помещениях UNION, не могут получить доступ к веб-сайтам UNION, системам обучения или другому программному обеспечению в среде XP.

Веб-сайты и учебные системы UNION, использующие веб-браузеры, будут продолжать работать, если пользователи смогут поддерживать последнюю версию веб-браузера в Windows XP.

Студенты, использующие Windows XP и испытывающие проблемы с нашими онлайн-ресурсами или другим программным обеспечением, предоставленным для обучения, могут по-прежнему обращаться в службу поддержки UNION, чтобы сообщить о проблемах (поскольку проблемы могут быть не связаны с Windows XP), однако UNION не сможет решить абсолютно все существующие проблемы, если они непосредственно вызваны использованием Windows XP.

Apple также прекратила поддержку OS X 10.6 (и ниже), что означает, что UNION ограничен в способах решения проблем.
